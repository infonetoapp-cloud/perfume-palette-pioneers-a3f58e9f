# Collection Banner System

## Core idea

There is only one perfume bottle source image for the perfume collections:

Do not solve the collection banners by changing the bottle. Solve them by changing:

- materials
- lighting
- surface styling
- props
- crop
- color temperature
- scene composition

This means every perfume collection still uses the same real David Walker bottle, but each banner feels distinct because the world around the bottle changes.

## Visual system

Create a consistent premium series inspired by modern minimal ecommerce category banners.

Style requirements:

- wide horizontal category-banner layout
- airy, warm, premium, elegant
- background palette: ivory, oat, stone, sand, warm beige
- clean editorial lighting
- soft shadows only
- lots of breathing room
- products should feel expensive and calm, not loud
- keep a clear blank area on the left side for UI text overlay
- place the product composition mostly on the right half
- all banners should feel like one family when viewed together

Do not:

- redesign the bottle
- change the logo
- invent a different bottle shape
- make the label unreadable
- add fake text or random typography into the image
- make the image too dark, busy, glossy, or cluttered
- turn it into a poster or ad with embedded copy

## Why this works

Because the bottle is the same across the perfume catalog, the collection identity should come from the scent world around it, not from product-packaging differences. This creates a cleaner and more premium system than forcing fake bottle variation.
