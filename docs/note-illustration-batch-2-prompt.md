This batch is intentionally strict.

Generate only literal, real fragrance ingredients or botanicals.
Do not generate abstract mood concepts in this batch.
Do not create visuals for generic labels like `Fresh`, `Floral`, `Woody`, `Warm`, `Sweet`, `Aromatic`, or `Clean`.

The purpose of this batch is to expand the ingredient library without introducing mismatched visuals.
